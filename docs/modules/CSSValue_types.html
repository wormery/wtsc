unset

unset: "unset"
  • Defined in src/CSSValue/types.ts:26

CSS关键字 unset 可以分为两种情况，如果这个属性本来 有从父级继承的值（这个属性默认可以继承，且父级有定义）， 则将该属性重新设置为继承的值，如果没有继承父级样式， 则将该属性重新设置为初始值。换句话说，在第一种情况下 （继承属性）它的行为类似于inherit ，在第二种情况下 （非继承属性）类似于initial。它可以应用于任何CSS 属性，包括CSS简写属性 all 。
